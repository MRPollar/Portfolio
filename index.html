Tecnologias

HTML5

HTML é a sigla para "HyperText Markup Language" - Lingaugem de Marcação de Hipertexto.

O HTML é o componente base da web, ele permite a construção dos sites e a incerção dos conteúdos do site, como imagens, vídeos, textos e outos elementos por meio de hipertexto

JavaScript

JavaScript é uma linguagem de programação que permite a implementação de elementos complexos na página web.

Quando um site mostra ao usuário mais do que informações estáticas, o javascript está por trás desse elemento permitindo uma experiencia mais agradável e interativa ao usuário.

CSS3

CSS se trata de uma folha de estilo estilo que trabalha juntamente com HTML ou XHTML

O CSS ajuda a editar, alinhar, remover e trabalhar no espaço entre elementos de uma página.

VUE.JS

VUE.JS é um framework JavaScript progressivo de construção de interfaces de usuário.

Bootstrap

O Bootstrap é um framework front-end que fornece estruturas de CSS para criação de sites e aplicações responsivas de forma rápida e simples.

LESS

O LESS foi criado para otimizar o código CSS e sua tradução é "menos CSS".

O LESS nos permite declarar váriaveis, funções e mixins oq torna o CSS altamente dinâmico.

Git

O Git é um sistema de versionamento, muito utilizado no desenvolvimento de software.

Github

O GitHub é o maior repositório de dados do mundo, é uma plataforma utilizada em sua maioria por desenvolvedores e permite uma hopedagem prática de código-fonte e arquivos na nuvem.
